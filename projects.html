Academic Projects

RPC | COMP117 Distributed Systems

A Python program to generate stubs and proxies in C++ from a C++ header file for remote procedure calls.

File Copy | COMP117 Distributed Systems

A C++ program to copy a directory of files across an imperfect network. After reading the famous End-to-end Arguments in System Design paper, we began by build an end-to-end verification system to check whether files had been correctly copied. We then implemented the logic to copy the files in packets and attempt to recover from dropped or reordered packets. Arith | COMP40 An image compressor and decompressor program written in C which operates on .ppm image files, compressing RGB pixel data into 32-bit words, and unpacking 32-bit words into RGB pixel data. Through this project, I studied bit-level representations and manipulations of data and floating-point arithmetic. Our final result compressor and decompressor both "produce less error than the reference implementation". Universal Machine | COMP40 An emulator for a Turing-complete universal machine (UM) with 13 commands written in C. This was a multi-part project: we spent the first week designing and building the UM, primarily focused on how best to emulate its segmented memory in software. We then spent the next week using profiling tools to optimize our UM to be more than 10x faster. Finally, we wrote a simple RPN calculator in UM assembly to run on the UM.
